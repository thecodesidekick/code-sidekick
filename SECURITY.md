# Security Policy

## Supported Versions

Currently, only the latest release of the Code Sidekick extension is supported with security updates. Please ensure you keep your extension updated via the Chrome Web Store.

## Reporting a Vulnerability

Security is a top priority for us. If you discover a security vulnerability within Code Sidekick, please do not disclose it publicly on the issue tracker.

Instead, please send an email to **support@codesidekick.dev**.

We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress. If you report a vulnerability that significantly affects the safety of our users, we may offer a bounty as a token of our appreciation.
