Welcome to the **Code Sidekick** community repository! While the core Code Sidekick engine remains closed-source, this repository serves as our central hub for transparency, feature planning, and community bug tracking.

## 🐛 Bug Reports
Found something strange? [Open a new Issue](https://github.com/thecodesidekick/code-sidekick/issues/new/choose) and select **Bug Report**. Please provide as much detail as possible so we can squash it quickly!

## 💡 Feature Requests
Have a brilliant idea that would speed up your workflow? Head over to our [Discussions Board](https://github.com/thecodesidekick/code-sidekick/discussions) and create a highly detailed feature request. The more upvotes it gets from the community, the faster it gets built.

## 🔒 Private Support
If you have questions about billing, enterprise plans, or need help with your specific account, please do not open a public issue. Drop us an email at **support@codesidekick.dev**.
